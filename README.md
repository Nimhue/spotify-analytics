# Spotify Analytics
## From Raw Data to Visual Insights

**Project #1** in the Data Engineering series.

A hands-on, step-by-step guide to building a complete analytics pipeline: ingesting streaming data, transforming it through a medallion architecture (Bronze → Silver → Gold), and surfacing insights with interactive visualizations.

## Live Demo

[View the article](https://nimhue.github.io/spotify-analytics/)

## What's Inside

- **Medallion Architecture**: Bronze, Silver, and Gold layer walkthrough
- **PySpark + SQL**: Production-style ingestion and transformation code
- **Interactive Charts**: Top artists, correlation heatmaps, mood quadrants
- **Audio Feature Analysis**: Danceability, energy, valence, and more
- **5 Key Insights**: What the data actually reveals about streaming hits

## Tech Stack

- PySpark / Delta Lake
- SQL (Databricks-compatible)
- HTML / CSS / JavaScript (article & visualizations)
- Medallion Architecture pattern

## Dataset

Based on the [Most Streamed Spotify Songs](https://www.kaggle.com/datasets/nelgiriyewithana/most-streamed-spotify-songs-2024) dataset from Kaggle.
